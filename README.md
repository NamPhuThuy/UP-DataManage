# Instruction

Available for Unity 6000.3.9f1

The package should be placed like this:
<pre>
_Project/
├── Module Data/
│   ├── Extend/
│   └── UPack-DataManage/
</pre>


# Dependencies
- https://github.com/NamPhuThuy/UPack-Common
- Unity package: In-App Purchasing (or delete the IAP-Data part)

# Note
